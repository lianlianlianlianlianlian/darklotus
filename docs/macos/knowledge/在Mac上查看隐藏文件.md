在Mac上查看隐藏文件，你可以使用以下快捷方式：

1. **打开Finder**：点击桌面或打开一个Finder窗口。
2. **使用快捷键**：按下 `Command (⌘) + Shift (⇧) + . (句点)`。这将切换隐藏文件的可见性。

再次按下相同的快捷键可以隐藏文件。

如果你想要在终端中查看隐藏文件，可以使用以下命令：

```bash
ls -a
```

这个命令会列出当前目录下的所有文件，包括以点（.）开头的隐藏文件。
