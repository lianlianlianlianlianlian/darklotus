1，在开始设置之前禁用互联网连接，例如，通过断开 LAN 电缆或禁用 Wi-Fi。

2，在屏幕上，使用 Shift-F10 打开命令提示符窗口。

3，在命令提示符窗口，输入 oobe\bypassnro 并按回车键。

4，Windows 将自动重新启动并返回到“让我们为你连接到网络”屏幕。只有这一次，你可以选择“我没有Internet连接”来跳过此操作。
